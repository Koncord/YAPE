YAPE
====
YAPE - Yet Another Processor Emulator.  
License: [Apache 2.0](LICENSE).  
##Requirements
All projects required CMake >= 2.8
###Core
* Boost >= 1.50  
* GCC >= 4.9

###Test Suite
* Boost >= 1.50
* GCC >= 4.9
* GTest (Google C++ Testing Framework) >= 1.7

###FontEditor
* QT >= 5.0
* GCC >= 4.9 or CLang 3.6
* Boost >= 1.50

###Plugin BaseIO
* SDL >= 2.0
* GCC >= 4.9 or CLang 3.6
